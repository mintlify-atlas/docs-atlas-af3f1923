The `Csfd` class is the primary interface for accessing ČSFD data. It provides methods to fetch movies, search content, retrieve creator information, and access user data.

## Import

```typescript
import { csfd } from 'node-csfd-api';

// Or create a custom instance
import { Csfd } from 'node-csfd-api';
```

## Default Instance

The library exports a pre-configured `csfd` instance that you can use immediately:

```typescript
import { csfd } from 'node-csfd-api';

const movie = await csfd.movie(535121);
```

## Constructor

While you typically use the default `csfd` instance, you can create a custom instance with default options:

```typescript
new Csfd(
  userRatingsService,
  userReviewsService,
  movieService,
  creatorService,
  searchService,
  cinemaService,
  defaultOptions?
)
```

<ParamField path="defaultOptions" type="CSFDOptions" optional>
  Default options applied to all API calls
  
  <Expandable title="properties">
    <ParamField path="language" type="'cs' | 'en' | 'sk'" optional>
      Preferred language for content (default: 'cs')
    </ParamField>
    
    <ParamField path="request" type="RequestInit" optional>
      Custom fetch request options (headers, timeout, etc.)
    </ParamField>
  </Expandable>
</ParamField>

## Methods

### setOptions()

Update the default options for all subsequent API calls.

```typescript
csfd.setOptions({
  language: 'en',
  request: {
    headers: {
      'User-Agent': 'MyApp/1.0'
    }
  }
});
```

<ParamField path="options" type="CSFDOptions" required>
  Options to set
  
  <Expandable title="properties">
    <ParamField path="language" type="'cs' | 'en' | 'sk'" optional>
      Preferred language for content
    </ParamField>
    
    <ParamField path="request" type="RequestInit" optional>
      Custom fetch request options
    </ParamField>
  </Expandable>
</ParamField>

## Example

```typescript
import { csfd } from 'node-csfd-api';

// Set default language
csfd.setOptions({ language: 'en' });

// All methods will now use English when available
const movie = await csfd.movie(535121);
const results = await csfd.search('Tarantino');
const creator = await csfd.creator(2120);
```
